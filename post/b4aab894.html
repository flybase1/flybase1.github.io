5. 核心概念

1. 路由 route

根据什么断言条件转发到哪里

2. 断言 Predicate

一组规则，条件如何转发路由

3. 过滤 filter

对请求集中处理

  1. 客户端发送请求
  2. Handler Mapping：根据断言转发请求
  3. Web Handler： 实际处理请求（经过过滤器）
  4. Service：调用服务

方式

  1. 配置式（推荐）
    1. 简化版： shortcut
    2. 完全版： Fully
  2. 编程式

5. 使用

5.1 断言Predicate

5.1.2 Between 在一段时间内

5.1.4 cookie route 请求cookie

5.1.5 Header route 请求头

5.1.7 Weight route 权重断言 （灰度发布）
